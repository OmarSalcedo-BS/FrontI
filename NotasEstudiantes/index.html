Ejercicio de desempeño

Ejercicio #2: Una universidad necesita un programa en JavaScript para calcular las notas finales de sus estudiantes en una materia.
El programa debe cumplir con lo siguiente:
1. Ingreso de datos
• Solicitar al usuario:
o Nombre del estudiante.
o Documento del estudiante.
o Cantidad de notas que se van a registrar (máximo 5).
• Para cada nota, el usuario debe ingresar un número entre 0 y 5.
o Si el valor ingresado no es un número o no está en el rango válido, se debe volver a pedir.

2. Cálculos
• Crear una función que calcule el promedio de las notas.
• Crear una función que determine el desempeño del estudiante según la nota final:
o 0 – 2.9 → Insuficiente
o 3.0 – 3.9 → Aceptable
o 4.0 – 4.6 → Sobresaliente
o 4.7 – 5.0 → Excelente
• Crear una función que devuelva un mensaje de aprobación o reprobación, sabiendo que se aprueba con 3.0 o más.
3. Impresión de resultados
• El programa debe imprimir en consola un reporte con el siguiente formato:

Reporte académico
-------------------------
Nombre: [nombre]
Documento: [documento]
Notas registradas: [n1, n2, n3, ...]
Promedio final: [valor]
Desempeño: [categoría]
Estado: [Aprobado / Reprobado]
4. Extras
• El programa debe permitir registrar la información de varios estudiantes (mínimo 3), usando un ciclo.
• Al final, se debe mostrar en consola:
o El nombre del estudiante con el mejor promedio.
o El nombre del estudiante con el peor promedio.
